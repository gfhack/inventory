# Inventory - Projeto de Pós Graduação em Desenvolvimento Web e Mobile - Guairacá 2019

## This project uses:
 - Vue
 - Vue Router
 - Vuex
 - Vue ChartJs
 - TailwindCss
 - Nes Css

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Instantiate the local database
```
json-server --watch db.json
```
